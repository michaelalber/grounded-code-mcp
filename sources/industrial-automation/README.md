# `industrial_automation` — Industrial Automation & OT

PLC programming, SCADA, industrial protocols, OPC UA, Raspberry Pi/edge
hardware, and ICS security. Used for industrial IoT and automation projects.

## What belongs here

- PLC programming and automation handbooks
- OPC UA specification documents
- MODBUS and industrial protocol references
- Raspberry Pi and single-board computer guides
- ICS/OT security (NIST SP 800-82)
- Robotics and motion control references
