# `4d_legacy` — 4D Platform Documentation

Reference documentation for the 4D database/IDE platform. Used as the
**source-side grounding** for 4D → .NET migration work. 4D is a niche
proprietary platform with minimal LLM training data coverage — grounding
in authoritative docs here is critical for accurate migration output.

Pair with `dotnet_csharp` for target-side patterns.

## What belongs here

- 4D language reference (variables, commands, syntax)
- 4D object model and form event documentation
- 4D v18, v19, v20+ version-specific docs
- 4D Internet Commands, 4D Write, plugin APIs
