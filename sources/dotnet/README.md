# `dotnet_csharp` — .NET / C# APIs & Frameworks

Reference documentation for the .NET ecosystem: language APIs, frameworks,
migration guides, and architectural guidance. 4D legacy docs live in the
sibling `4d-legacy/` collection, not here.

## What belongs here

- .NET runtime and BCL documentation
- ASP.NET Core and web framework references
- Entity Framework Core
- Dependency injection in .NET
- Migration guides (.NET Framework → .NET 8/10)
- Azure and cloud integration for .NET
